\section{Introduction}
Reconstructing missing EEG channels is a common problem in neurophysiology and wearable sensing. Graph Signal Processing (GSP) provides a principled framework to exploit spatial relations among sensors for interpolation and reconstruction \cite{shuman2013,narang2013,puy2018}. Recent advances propose time-aware priors and TV-based reconstructions that leverage temporal smoothness in addition to graph structure \cite{giraldo2022,zhang2024bgsrp}.

This work focuses on an empirical, reproducible comparison executed inside the repository pipeline. We rely exclusively on generated artifacts (CSV tables, per-iteration PDFs and run metadata) and we follow the project's traceability policy: every quantitative claim is anchored to a results file and the generating script (for example \showresfile{experiments/run_canonical_experiment.py} and the consolidated editorial traceability \showresfile{results/b3_b4_editorial_traceability.md}). The project executed a broad iteration campaign (it02–it130); the comprehensive inventory and per-phase summaries are available in the repository report \showresfile{Thesis-Copilot-Toolkit/ITERATIONS_COMPREHENSIVE_REPORT.md}.

INS-13 editorial constraints are applied consistently: cross-stack MATLAB/GSPBox checks are included as controlled proxy comparisons and declared as \texttt{proxy\_or\_partial} with \texttt{strict\_close=False} (canonical status: \showresfile{results/ins13_strict_status.md}).
