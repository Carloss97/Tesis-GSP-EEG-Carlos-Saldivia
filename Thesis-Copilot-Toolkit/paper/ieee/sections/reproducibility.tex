\section{Code Availability}

The complete source code, including all interpolation algorithms, graph constructors, evaluation metrics, Optuna optimization scripts, and figure generation pipelines, is publicly available at \url{https://github.com/Carloss97/Tesis-GSP-EEG-Carlos-Saldivia}. The repository includes frozen configuration files and deterministic random seeds (\texttt{random\_state=42}) to ensure full reproducibility of the reported results.
