\section{Code Availability}

The complete source code, including all interpolation algorithms, graph constructors, evaluation metrics, Optuna optimization scripts, and figure generation pipelines, is available at an anonymous repository for review: \url{https://anonymous.4open.science/r/EEG-GSP-Interpolation-A1B2} (Placeholder for review). The repository includes frozen configuration files and deterministic random seeds (\texttt{random\_state=42}) to ensure full reproducibility of the reported results.
