\section{Reproducibility Statement}

To support reproducibility, this project maintains experiment scripts, frozen configuration files, raw outputs, and aggregated summaries under version control. Each reported table or figure should be traceable to: (i) a script in the experiments directory, (ii) a configuration artifact, and (iii) generated CSV/figure files.

Before submission, we will release the final command list used to generate manuscript figures and tables, including dependency versions and random seeds.
